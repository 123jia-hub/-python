# 学生成绩管理系统 (Student Score Analyzer)

一个基于 Python 的学生成绩管理系统，支持添加学生信息、自动计算分数、数据持久化存储等功能。


## 项目简介

本项目旨在提供一个简单易用的学生成绩管理工具，帮助用户快速录入、管理和分析学生成绩数据。系统采用面向对象设计，数据存储使用 JSON 文件，确保数据持久化和跨会话保存。


## 功能特点

- **添加学生信息**：输入学号、姓名、语文/数学/英语成绩，系统自动验证输入合法性。
- **自动计算分数**：自动计算学生的总分和平均分。
- **数据持久化**：将学生信息保存到 JSON 文件，程序重启后数据不丢失。
- **输入验证**：确保成绩在 0-100 之间，非数字输入会被提示重新输入。
- **异常处理**：捕获并处理可能的错误，确保程序稳定运行。


## 技术栈

- **编程语言**：Python 3.x
- **数据存储**：JSON 文件
- **标准库**：`json`、`os`、`datetime`


## 使用方法

### 1. 克隆仓库
```bash
git clone https://github.com/你的用户名/student-score-analyzer.git
cd student-score-analyzer
```

### 2. 运行程序
```bash
python student_score_analyzer.py
```

### 3. 操作流程
1. 程序启动时会自动加载 `student_data.json` 文件中的现有学生数据。
2. 按照提示输入学生信息：
   - 学号（确保唯一）
   - 姓名（不能为空）
   - 语文、数学、英语成绩（0-100 之间的有效数字）
3. 输入完成后，系统会自动计算总分和平均分，并将数据保存到文件。
4. 程序会显示添加成功的提示信息，包括学生姓名、总分和平均分。


## 数据文件格式

学生数据以 JSON 格式存储在 `student_data.json` 文件中，结构如下：
```json
[
  {
    "id": "1001",
    "name": "张三",
    "scores": {
      "语文": 85.0,
      "数学": 90.0,
      "英语": 95.0
    },
    "total": 270.0,
    "avg": 90.0
  }
]
```
